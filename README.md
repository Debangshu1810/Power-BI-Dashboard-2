# Power-BI-Dashboard-2
🛒 BLINKIT Sales Analysis Dashboard
🚀 End-to-End Power BI Project | Data Analytics Portfolio
Project Overview:
An interactive Power BI dashboard built to analyze Blinkit's sales performance across item categories, outlet types, and locations.
This project demonstrates data cleaning, modeling, DAX calculations, and storytelling through dashboards to generate actionable business insights.

🚀 Problem Statement
Businesses like Blinkit operate across multiple cities and outlet formats. Understanding Which products drive the most revenue or
Which outlet types perform best Customer preferences and trends is critical for growth and optimization.

📖 Objective:
To build a dashboard thatTracks total and average sales performance and Identifies top-performing product categories that Analyzes outlet size and location impact
Provides clear, decision-ready insights,
🛠️ Tech Stack
📊 Power BI – Dashboard & Visualization
🧮 DAX – Measures & KPIs
📂 Excel / CSV – Data source
🔗 Data Modeling – Relationships & transformations
📂 Dataset Details

The dataset includes:
🥦 Item categories (Fruits, Snacks, Dairy, etc.)
🏬 Outlet size (Small, Medium, High)
🌆 Outlet location (Tier 1, Tier 2, Tier 3)
📅 Establishment year
⭐ Ratings & sales values
⚡ Key Features

✔️ Interactive slicers for dynamic filtering
✔️ KPI cards for quick business overview
✔️ Category-level sales analysis
✔️ Outlet performance comparison
✔️ Customer preference insights (Fat content)

💡 Key Business Insights

📈 High-demand categories: Fruits & Vegetables and Snack Foods
🏬 Best performing outlet type: Medium-sized outlets
🌍 Location insight: Tier 3 cities show strong contribution
🥗 Customer trend: High preference for Low Fat products
📉 Opportunity: Improve product quality (ratings ~3.9)
🎯 Business Impact: This dashboard helps stakeholders to Optimize inventory planning, Focus on high-revenue categories, Improve outlet expansion strategies,  Understand customer behavior and Make data-driven decisions.
